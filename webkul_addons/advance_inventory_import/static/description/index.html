    Key Points To Follow while Bulk Update Inventory in Odoo With This Module

    How To bulk Update Product Inventory in Odoo using CSV file

    Use the following method to update the inventory

  • Create a csv file in your system.
  • Put the name of the column headers as mentioned in the Keys table below.
  • The name of the column should match exactly with the keys as shown in the table.
  • The system uses the column headers to find correct products in Odoo.
  • For example if you wish to find products with product external id use product_extermal_id as the column header.
  • One Column header containing the product key and the quantity column is mandatory in the CSV file.
  • You can add multiple columns with different headers to locate the correct products.Â 
  • The warehouse keys are used to find the specific warehouses in which the respective products are to be updated. If there is no column for warehouse in file, the system will update the product stock in the default selected warehouse in Odoo.
  • Similarly the product lot key is used to locate the product lot number/serial number whose quantity to be updated.
  • Create the file and simply import the file to Odoo.
  • The listed new quantities of each product in the file is added to the current product quantity in Odoo.
