Website Order Cancel

Odoo Website Order Cancel Module allows customers to cancel their orders from their Odoo Website account. The Customer can click on cancel order button next to the order quotation to cancel the order.

Salient Features
  • Allows to cancel order from website.
  • Manage Pre-defined order cancellation reason
  • Specify Cancellation reason while cancel order
  • Enable/Disable Order cancellation feature in Quotaiions.
