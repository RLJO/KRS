Odoo Stock Picking Cancel

This module helps to cancel the done stock pickings.

Basic Features
  • This module helps to cancel the done stock pickings.
  • Also, cancel stock moves and stock quants..
  • Introduce a new group named 'Allow Done Pickings Cancellation'.
  • Create Predefined reasons to cancel stock picking.
  • Cancel stock picking with detailed reason.
  • Picking cancellation mail notification will be sent to the followers.
