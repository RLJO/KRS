Odoo Marketplace Mass Upload

Note : The admin can see the product and Inventory upload history of all the sellers and The sellers can see the records of their uploads only in their seller profile in the Odoo backend.

Reduced efforts in Product and Inventory management on Your Marketplace

Instantly add products to odoo marketplace in bulk with a simple zip containing images folder and a csv file upload and Also update inventory of the desired product via single csv file.

The added products are automatically created in odoo.

Allow Sellers to Bulk Upload New Products to Odoo Marketplace

The module helps the Odoo Marketplace Sellers to create new products in bulk for their shop.

The imported products are auto created in the Odoo with status of 'Approved' or 'To Approve' depending upon the settings by admin.

Directly Update the Inventory with a single step with a CSV File

Prevent mannual inventory update of each product on Odoo Marketplace.

The module allows the sellers to update inventory of their Marketplace products in bulk with a CSV File Upload.

Each seller can update the inventory of his/her product(s).

The update request is either approved automatically or goes into 'To Approve' state depending upon the marketplace admin settings.

KEY FEATURES

Below is the detailed list of Feature for Odoo Marketplace Mass Upload Module

Bulk Import Products to Odoo Marketplace

  • The seller can import marketplace products using zip file.
  • The zip contains a product CSV file and image folders.
  • The imported products are auto-created in Odoo.
  • The status of added product depends upon admin settings - approved or to approve.

Bulk Update Marketplace Product Inventory

  • The sellers can bulk update inventory of their products with CSV file.
  • The CSV file includes the required fields and data.
  • The status of inventory update depends upon admin settings - approved or to approve.

Set the rules for creating CSV files

  • Set the required fields in CSV file for uploading products.
  • The zip contains a product CSV file and image folders.
  • If the required field(s) is missing the upload would go unsuccessful.

Multi Website Compatible

  • The Module is designed to be multi website compatibility.
  • The admin can create independent uploading rules for each website.

Track every upload Operation

  • The history of each upload operation is stored in Odoo.
  • The admin can see the history of every seller.
  • The sellers can see only their upload history.

Instantly check for Unsuccessful uploads

  • The module lists the unsuccessful records (if any) of each operation.
  • The admin can create independent uploading rules for each website
