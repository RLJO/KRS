Odoo Maintenance Mode

Odoo Maintenance Mode allows to put Odoo server(both website and backend) in the maintenance mode while performing some diagnostic tests on server. It allows to prevent customer requests during maintenance mode.

Salient Features

  • Put your odoo in maintenance mode with a simple click.
  • Put both Odoo backend and Website in maintenance mode.
  • Allow access to specific users during maintenance mode by specifying their IP addresses.
  • Restrict all customers from accessing your Odoo whose IP addresses are not specified.
  • Admin has always full access during maintenance mode.
  • A login button is displayed and only restricted users can login during maintenance mode.
  • A custom responsive page will be dispalyed to all the restricted users during maintenance mode.
  • All the messages displayed during maintenance mode are editable, customizable and translatable.
  • Image displayed during maintenance mode is customizable.
  • Display also translatable and customizable messages to the accessible users on your choice.
  • Allow your customers to enter proper validated emails for notifications during maintenance mode.
  • Send email notifications to the users who opted for the notification after maintenance mode is over.

Configuration and website view

Customizable & Responsive View

All the messages displayed are translatable and customizable. Customizable images can be displayed in the page. A notification feature is also displayed that allows the customers to enter their emails. Proper messages after success/error of entering of an email. A login button is also displayed for login of the accessible users. The page displayed is fully responsive.

Login Page Maintenance Mode

A login page displayed for login of the accessible users. All the non-accessible users are restricted to login during the maintenance mode. A proper customizable and tranlatable message displayed for non-accessible users during maintenance mode during login.

Messages displayed to accessible users

Proper customizable and translatable messages to the accesible users will be displayed in Backend and Website. Messages can be Enabled/Disabled.

Set Default Configuration

Enable/Disable the maintenance mode on a single click.Configure your translatable messages to be displayed during maintenance mode. Specify the comma(,) separated IPs for which you want to give the access during maintenance mode. Admin will be accessible by default. Enable/Disable notification in the page displayed in the maintenance mode. All the emails entered by users during maintenance mode will be displayed with states. Send email notifications to the pending after being back in working mode.
