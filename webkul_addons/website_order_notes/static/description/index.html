Website Order Notes

Add Extra fields in your website for the ease and exact order delivery. Deliver the orders to your customers only when they are available.

Salient Features

  • Allows to leave a message on confirming an order from website.
  • Allows to specify a date for order delivery.
  • Restrict users before and after some specified number of days.
  • Allows To leave a message on confirming an order from odoo.
  • Manage messaging in stock picking.
  • Manage the messaging through all ways of invoicing also.
