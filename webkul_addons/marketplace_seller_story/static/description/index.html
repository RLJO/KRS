Odoo Marketplace Seller Story
Publish success story of your marketplace sellers

Note: This module is an addon for Odoo Multi Vendor Marketplace module. The price reflected here includes all dependent module's cost as well.

Features:

  • Admin/Seller can create seller story
  • A seller can publish Youtube video for his success story
  • Admin/Seller can publish or unpublish story at any time
  • Admin can mark seller stories as a popular story
  • Admin can show seller story carousel on any web page using seller story snippet
  • A visitor will be able to see seller stories on a separate story page on the website

  • Website View

  • Back End View
